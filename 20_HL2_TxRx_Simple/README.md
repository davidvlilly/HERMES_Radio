# HL2 Spot Test

A small Hermes-Lite 2 toolset for transmitting and detecting a weak CW "spot"
signal between two stations, built on the HPSDR Protocol 1 interface.

## Result

**A CW spot was successfully transmitted from San Diego to Fullerton (~40 m / 7 MHz)
and received at ~20 dB SNR.**

This validated the full chain: HL2 transmit (PA enabled, drive level set), the
over-the-air path, and the narrow-RBW receiver display needed to pull a weak CW
tone out of the noise.

## Files

| File | Description |
|------|-------------|
| `main_ex1.py` | Main application. Connects to the HL2, runs the live spectrum display, and drives transmit (PTT, En PA, TX-power slider), LNA gain, carrier frequency, RX calibration, forward-power telemetry, and the 100 ms IQ Capture button. Run: `python main_ex1.py` |
| `spectrum_ex1.py` | Spectrum engine + GUI. Welch power-spectrum computation and the live matplotlib window: PTT / En PA / Overlay toggle buttons, LNA and Freq entry boxes, TX-power slider, Capture button, and the PEP / PA-current power readout. |
| `view_time.py` | Offline time-domain viewer for `capture.npz`. Plots I/Q and envelope in volts with average power, and applies a ±2 kHz band-pass around the frequency of interest. Run: `python view_time.py` |
| `HL2_Block_Dgm.pdf` | Hermes-Lite 2 hardware block diagram (reference). |
| `capture.npz` | Generated by the Capture button — 100 ms of raw IQ for `view_time.py`. |

## Key settings (in `main_ex1.py`)

- `NCO_FREQ` — carrier / receive frequency (default 7.000 MHz; also adjustable live via the **Freq** box).
- `SAMPLE_RATE` 384 ksps, `SEGMENT_SIZE` 32768 → **RBW ≈ 17 Hz** (matches Quisk; narrow enough to see weak CW).
- `DBFS_REF_DBM = 8.0` — RX power calibration (ADC-referred / after LNA).
- `TX_AMPLITUDE = 0.8` — digital DAC level; actual output power set by the **TX Pwr** slider (0–15) and **En PA**.

## Transmitting a spot

1. Run `python main_ex1.py`.
2. Set the carrier (**Freq** box) and LNA gain as needed.
3. Enable **PTT** and **En PA**, set the **TX Pwr** slider for the desired output.
4. Watch the **PEP / PA mA** readout to confirm power is going out.
5. The far station sees the tone as a peak on its spectrum; use the ±2 kHz band-pass
   in `view_time.py` on a capture to measure it cleanly.
